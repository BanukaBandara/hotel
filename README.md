# Grand Terrace Restaurant Website (React + Tailwind)

A clean, luxury-inspired restaurant website template (original design) built with:
- React (Vite)
- Tailwind CSS
- React Router
- Framer Motion
- Lucide Icons

## 1) Install
```bash
npm install
```

## 2) Run locally
```bash
npm run dev
```

## 3) Build for production
```bash
npm run build
npm run preview
```

## Customize quickly
- Venue list: `src/data/venues.js`
- Contact details: `src/pages/Contact.jsx`
- Reservation email target: `src/components/ReservationModal.jsx`
- Logos: `src/assets/`
